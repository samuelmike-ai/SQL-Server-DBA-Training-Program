### **STEP 2: Create New Repository**
1. **Click:** Green "New" button (or "+" icon ‚Üí "New repository")
2. **Repository name:** `SQL-Server-DBA-Training-Program`
3. **Description:** `Complete 6-Month SQL Server DBA Training Program - Created with MiniMax Agent using IGR Framework`

5. **Add README:** ‚úÖ Check "Add a README file"

7. **Choose license:** Select "MIT License"
8. **Click:** Green "Create repository" button

### **STEP 3: Upload Your Program Files**
1. **Click:** "uploading an existing file" link
2. **Drag and drop** all files from your training program:
   - Master program document
   - All weekly curriculum files

   - Templates and methodology
   - Flexibility guide
   - Social media content
3. **Commit message:** `Initial program upload - Complete SQL Server DBA training program`
4. **Click:** Green "Commit changes" button

1. **Add folders** by creating new files with folder paths:
