### ğŸ› ï¸ Step-by-Step Setup

#### **Step 1: Download SQL Server (30 minutes)**
1. Open your web browser (Chrome, Edge, Firefox)
2. Go to: `microsoft.com/sql-server`
3. Click "Download SQL Server 2019 Developer Edition"
4. Choose "Developer Edition" and click "Download now"
5. Wait for download to finish (about 1GB)

#### **Step 2: Install SQL Server (45 minutes)**
1. Find the downloaded file (usually in Downloads folder)
2. Double-click the file to start installation
3. Choose "New standalone installation"
4. Accept the license agreement
5. Select "Developer" edition (it's FREE!)
6. Click "Next" through all options (keep defaults)
7. Wait for installation (this takes 20-30 minutes)
8. Click "Close" when done

#### **Step 3: Download Management Studio (15 minutes)**
1. Go to: `docs.microsoft.com/sql/ssms/`
2. Click "Download SQL Server Management Studio"
3. Wait for download to finish (about 800MB)

#### **Step 4: Install Management Studio (30 minutes)**
1. Find the downloaded SSMS file
2. Double-click to start installation
3. Click "Install"
4. Wait for installation (about 20 minutes)
5. Click "Close" when finished

#### **Step 5: Test Your Setup (10 minutes)**
1. Press Windows key + R
2. Type "SSMS" and press Enter
3. When SSMS opens, click "Connect"
4. Server name should be "(local)" or your computer name
5. Click "Connect"
6. You should see "Object Explorer" with server info!
